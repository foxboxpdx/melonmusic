# MelonMusic
Simple music suggestion and review web app in Rust

## What
* Axum + Diesel backend, Yew + Wasm-Pack frontend
* More-or-less anonymous because I can't be bothered to build an authentication system
* Intended for small-ish friend groups / Discord servers / etc.
* Add links to cool music you want to share, keep track of what friends have posted, whether you've checked it out yet, whether you thought it was any good, etc.
* Lots of potential for additional functionality but ultimately as simple as possible

## Why
* Bored

* Roundabout way to 'export' knowledge gained from work projects
